# Machine Learning - Adaptación de Material Docente

Este repositorio contiene código y ejemplos prácticos relacionados con **Machine Learning** y **Pronósticos en los Negocios**.  
El contenido aquí presentado **no es original**, sino una **adaptación y actualización** del material compartido por el profesor **Julio César Galindo Pérez** en el curso.

## Objetivo
- Reforzar los conceptos v  mediante la práctica en código.
- Aplicar modelos de ML.
- Facilitar el aprendizaje mediante ejemplos reproducibles en Python.

## Nota Importante
El código aquí incluido:
- Es únicamente con fines **educativos**.
- No debe considerarse como un desarrollo propio, sino como una **adaptación** del material proporcionado por el profesor.
- Puede contener modificaciones menores para ajustarse al entorno de ejecución personal (por ejemplo, rutas de archivos, librerías, o formato de salida).

## Contenido
- Scripts de regresión y pronósticos.
- Ejemplos de series de tiempo.
- Ejercicios de práctica basados en datasets de clase.
